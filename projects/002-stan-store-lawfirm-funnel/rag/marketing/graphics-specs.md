## Brand Guidelines Reference

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Dark Background | `#0A0A14` | Primary background |
| Pure Cyan | `#00FFFF` | Accent, key metrics, glow effects |
| White | `#FFFFFF` | Headlines, primary text |
| Gray (Secondary) | `#B0B0B0` | Subtext, secondary info |
| Cyan Glow | `rgba(0, 255, 255, 0.3)` | Soft glow behind key elements |

### Typography
| Element | Font | Weight | Size | Color |
|---------|------|--------|------|-------|
| Headline Number | Playfair Display | Bold (700) | 72px | `#00FFFF` |
| Headline Text | Playfair Display | Regular (400) | 36px | `#FFFFFF` |
| Subtext | Inter | Regular (400) | 24px | `#B0B0B0` |
| Source | Inter | Light (300) | 16px | `#808080` |
| CTA | Inter | SemiBold (600) | 18px | `#00FFFF` |
| URL | Inter | Regular (400) | 14px | `#666666` |

### Visual Style
- Retro-futurist aesthetic
- Neon glow effects on cyan elements
- Subtle grid landscape texture in background
- Dark, premium feel
- Clean, minimal layout with breathing room

### Dimensions
- Width: 1200px
- Height: 628px
- Format: PNG (final), SVG (mockup)
- Resolution: 72 DPI (web optimized)

### Content

### Spacing
- Top padding: 80px
- Left padding: 80px
- Right padding: 80px
- Bottom padding: 60px
- Line height: 1.3 for headlines, 1.5 for body
- Gap between headline and metric text: 8px
- Gap between metric and subtext: 40px
- Gap between subtext and source: 20px
- Gap between source and CTA: 48px

### Effects
- Headline "2-3 hours saved":
  - Color: `#00FFFF`
  - Text shadow: `0 0 30px rgba(0, 255, 255, 0.5)`
  - Additional glow: `0 0 60px rgba(0, 255, 255, 0.25)`
- Background grid:
  - Lines: `rgba(0, 255, 255, 0.05)`
  - Perspective tilt toward horizon
  - Gradient fade from bottom
