## Primary Wordmark

### Design Specifications

```
Text:           BOWTIE BULLIES
Subtitle:       THE AFTERMATH
Font:           Bangers (Google Fonts)
Weight:         400 (Regular)
Case:           ALL UPPERCASE
Letter-spacing: +3% (intentional breathing room)
Color:          #E7E7E1 (Bone White) on dark backgrounds
                #1E1E20 (Dark Steel) on light backgrounds (rare)
Accent:         "BULLIES" in #9A4C22 (Rust Orange) — optional variant

### Wordmark Variants

| Variant | Layout | Use |
|---------|--------|-----|

### Subtitle Treatment

```
"THE AFTERMATH"
Font:           Montserrat
Weight:         400
Case:           ALL UPPERCASE

## BowTie Mark (Brand Emblem)

### Design Concept

The Chevrolet bowtie silhouette — a horizontal diamond-cross shape. NOT a literal butterfly bowtie. This is the most recognizable micro-element of the brand. It appears on channel branding elements and on vehicles. NEVER placed on Red Nose's body.

### Specifications

```
Shape:          Chevrolet bowtie silhouette (horizontal diamond-cross)
Style:          Clean geometric edges, matte finish — NOT weathered, NOT fabric
Fill:           #1E1E20 (Dark Steel / Matte Black)
Trim:           #8B0000 (Deep Crimson) — ultra-thin border, 3-5% of height
Alt Fill:       #E7E7E1 (Bone White) — on dark backgrounds (alternate)
Minimum size:   24x24px (digital), 0.5" (print)
Clear space:    1x mark width on all sides
```

### Mark Usage

| Context | Size | Color | Notes |
|---------|------|-------|-------|
| YouTube watermark | 150x150px | Dark Steel fill + Crimson trim on transparent | Persistent corner element |
| Favicon | 192x192px | Dark Steel fill + Crimson trim | Browser tab, bookmarks |
| App icon | 1024x1024px | Dark Steel fill + Crimson trim on Dark Steel bg | If needed for future apps |
| Bullet point | 16-24px inline | Dark Steel fill + Crimson trim | In descriptions, lists |
| Divider element | 32px centered | Ash Gray fill | Between sections in video |
| Vehicle grille emblem | Relative to vehicle | Matte black + Crimson trim | Always present on vehicles |

### BowTie Mark Rules
1. **NEVER a literal fabric bowtie.** No butterfly shape, no collar placement.
2. **Always the Chevrolet silhouette** — clean, geometric, recognizable.
3. **Deep crimson trim only** — never bright red, never neon, never glowing.
4. **Matte finish always** — no shine, no gloss, no metallic reflection.
5. **≤5% of visual frame** when used in video/imagery.
6. **NEVER placed on Red Nose's body.** The mark is environmental and vehicular only.

## Forbidden Uses

| Rule | Reason |
|------|--------|
| Never rotate the wordmark | Stability = the brand's energy |
| Never use gradients on text | No decoration. Just truth. |
| Never use drop shadow on wordmark | Shadows are for the environment, not the type |
| Never use decorative fonts | Bangers and Montserrat only (Anton as fallback) |
| Never place on busy/bright backgrounds | Dark Steel or clean dark surface only |
| Never stretch or distort | Proportions are sacred |
| Never add outlines or strokes to text | Clean, not embellished |
| Never use the icon without the Chevrolet bowtie silhouette | The BowTie Mark IS the icon |
| Never use bright accent colors | Rust Orange and Deep Crimson (BowTie Mark only) are the only accents |
| Never separate "THE AFTERMATH" from context | It always appears with or near the primary wordmark |

## File Naming Convention

```
btb-logo-[variant]-[color]-[size].[ext]

Variants: full, stacked, short, icon, banner, profile
Color: dark (white on dark), light (dark on light), mono, accent
Size: sm (small/web), md (medium), lg (large/print), xl (banner)
Extension: svg (vector), png (raster), pdf (print)

Examples:
btb-logo-full-dark-lg.svg
btb-logo-icon-dark-sm.png
btb-logo-stacked-accent-md.png
btb-logo-banner-dark-xl.png
btb-logo-profile-dark-md.png
```
