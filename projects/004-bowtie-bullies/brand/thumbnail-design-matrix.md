# BowTie Bullies — Thumbnail Design Matrix

> Every thumbnail is an argument in 3 seconds. If it doesn't make someone feel something — anger, curiosity, challenge — it's decoration.

---

## STEP 1: CONTENT MINING (Do This First — Before Any Design)

Before opening a single asset, answer these:

| Question | Purpose |
|----------|---------|
| What is the single most SHOCKING fact in this episode? | This becomes the headline |
| What emotion does it trigger? (anger, disbelief, curiosity, empowerment) | This determines color temperature + expression |
| What is the concrete value the viewer receives? | This is the secondary hook / banner |
| What "they" angle exists? (system vs viewer) | This determines confrontational framing |
| What would make someone TELL someone else about this? | This is the shareability test |
| Why would someone click NOW, not later? | This determines urgency language |

**Rule: If you can't fill this table, you don't understand the episode well enough to make a thumbnail.**

---

## STEP 2: HOOK CONSTRUCTION (Words Before Design)

### Primary Text (The Headline)
- **3-5 words MAXIMUM.** Not a sentence. A punch.
- Must create a **curiosity gap** — the viewer NEEDS to know the answer
- OR must trigger **anger/disbelief** — the viewer NEEDS to react
- Test: Would someone screenshot this and send it to a friend?

### Accent Word (The Payload)
- **ONE word** per text block in Hot Orange / brand accent
- This word carries the EMOTIONAL weight of the entire thumbnail
- It's the word that makes the viewer's brain stop scrolling
- Examples: "CRIME", "ILLEGAL", "NOTHING", "TAKE", "HIDDEN"

### Banner Text (The Second Hook)
- The banner is NOT a slogan. It's a **reason to click NOW.**
- It must add SPECIFICITY, URGENCY, or CHALLENGE
- Specificity: dates, numbers, laws, names → makes it REAL
- Urgency: "STILL HAPPENING", "BEFORE IT'S TOO LATE"
- Challenge: "NO EXCUSES", "ARE YOU READY?", "PROVE IT"
- **Test: Does the banner make sense if the viewer ONLY reads the banner?**

### Word Psychology Tiers
| Tier | Effect | Examples |
|------|--------|----------|
| **Anger words** | Trigger outrage, sharing | ILLEGAL, CRIME, STOLEN, RIGGED, BANNED |
| **Curiosity words** | Create information gap | HIDDEN, SECRET, EXPOSED, REAL REASON |
| **Challenge words** | Provoke the viewer | YOU CAN'T, PROVE IT, NO EXCUSES, ARE YOU? |
| **Value words** | Promise concrete knowledge | BUILD, OWN, FREE, BLUEPRINT, STARTING FROM |
| **Urgency words** | Demand immediate action | NOW, BEFORE, STILL, ALREADY, RUNNING OUT |

## STEP 3: CHARACTER SELECTION

| Criterion | Rule |
|-----------|------|
| Expression | Must match the emotional register. Angry content = intense face. Value content = confident/stoic. |
| Face visibility | **Face is NEVER covered by text.** The face IS the emotional hook. |
| Zone ownership | Character occupies its own visual zone. Text and character are ADJACENT, not stacked. |
| Scale | Character fills 50-70% of frame HEIGHT. Large enough to see expression at 160px thumbnail scale. |
| Eye contact | Direct-to-camera = confrontation. Off-frame = mystery/depth. Choose deliberately. |

### BowTie Character → Emotion Map
| Asset | Emotion | Best For |
|-------|---------|----------|
| Headshot (redNose_headshot) | Serious, direct, "listen to me" | Shocking facts, confrontation |
| Arms crossed | Authority, "I know something" | Value content, lists, ownership |
| Pointing at camera | Accusation, "this is about YOU" | System exposure, "they" content |
| Staring into distance | Contemplation, weight | Historical content, philosophical |
| Angry/teeth | Rage, intensity | Injustice, betrayal, outrage |
| Sunglasses variants | Cool authority, mystery | Confidence, unbothered power |

---

## STEP 4: BACKGROUND SELECTION

| Rule | Reason |
|------|--------|
| Scene TELLS A STORY | The background should communicate something BEFORE the viewer reads the text |
| Art style matches character | Cel-shaded backgrounds for cel-shaded character. No style mismatch. |
| Dimmed but VISIBLE | Background at 40-50% brightness. Must see detail. Not a black void. |
| No negative space | Every pixel carries information. Empty dark areas = wasted real estate. |
| Environmental props add density | Signs, objects, textures all make the thumbnail feel RICH |

---

## STEP 5: COMPOSITION (The Layout Law)

```
┌─────────────────────────────────────────────┐
│  TEXT ZONE (top 35%)                        │
│  [HEADLINE LINE 1]          ┌──────────┐   │
│  [HEADLINE LINE 2]          │CHARACTER │   │
│                              │  FACE    │   │
│                              │  ZONE    │   │
│                              │ (right   │   │
│  (bg scene visible here)     │  35-40%) │   │
│                              │          │   │
│ ▓▓▓▓ BANNER ▓▓▓▓            └──────────┘   │
│ (bottom 12%)                                │
└─────────────────────────────────────────────┘
```

### Rules:
1. Text and character occupy **ADJACENT** zones — never stacked
2. Character's face is in the **right third** (eye-tracking sweet spot)
3. Text fills the **left 55-60%** of the frame
4. Background scene is visible in the **gaps** — not hidden by a dark void
5. Banner sits at **bottom**, full width, paint-brush style
6. **Zero negative space** — if an area is empty, the background must carry it

---

## STEP 6: COLOR HIERARCHY (Brightness = Importance)

```
BRIGHTEST ──→ Accent word (Hot Orange #EB9B2D) ← viewer sees this FIRST
            → Primary text (Bone White #E7E7E1) ← viewer reads this SECOND
            → Character (rim-lit, warm glow)     ← viewer looks at face THIRD
            → Banner bar (Rust Orange + white text)
DIMMEST ───→ Background scene (40-50% brightness)
```

---

## STEP 7: BANNER STYLE

Banners are NOT plain rectangles. They are **paint-brush strokes** — textured, organic, with rough edges. This communicates:
- Raw authenticity (matches "Raw, Not Chaotic" voice pillar)
- Street-level energy (not corporate, not templated)
- Visual contrast against the clean text above

---

## QUALITY CHECKLIST (Before Export)

- [ ] Can I read the headline at 160px wide? (Phone search results)
- [ ] Is the character's FACE fully visible and unobstructed?
- [ ] Does the accent word make my brain STOP?
- [ ] Does the banner text make sense ON ITS OWN?
- [ ] Is there ANY dead/empty space? (There shouldn't be)
- [ ] Does the background scene support the story?
- [ ] Would I click this if I saw it while scrolling?
- [ ] Would someone send this thumbnail to a friend?
- [ ] Does this feel TRUE to the episode content?
- [ ] Is every element earning its space?
