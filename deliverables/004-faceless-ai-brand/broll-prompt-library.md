## How To Use This Library

Every prompt below is ready for direct input to Gemini Imagen, Midjourney, or DALL-E. Each entry includes:

- **Prompt ID** — Unique reference for pipeline tracking (BR-XX)
- **Generation prompt** — Full text, paste directly into your tool
- **Negative prompt** — Always include. Non-negotiable.
- **Episode tags** — Which episodes from `episodes.md` this supports
- **Content pillar** — AI & The System (P1) or The Bigger Picture (P2)
- **Mood tag** — For sorting by emotional tone
- **Aspect ratio** — 16:9 (landscape/YouTube) and/or 9:16 (vertical/Shorts/Reels)
- **Recommended platform** — Which AI tool produces best results for this type
