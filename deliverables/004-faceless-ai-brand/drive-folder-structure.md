## Folder Descriptions

### Top-Level Folders

| Folder | Purpose |
|--------|---------|
| **Character Sheets/** | Canonical Red Nose reference images for AI generation consistency. Organized by pose type (P1-P6). |
| **Music Library/** | Pre-curated royalty-free tracks organized by mood. Naming convention: `btb-music-{mood}-{nn}-{duration}s-128k.mp3` |
| **SFX Library/** | Sound effects for transitions, UI sounds, ambient loops, and stingers. |
| **Episodes/** | Per-episode production folders. All assets for a given episode live under its folder. |
| **Thumbnails/** | Final and draft thumbnails organized by season. |
| **Templates/** | Reusable overlay templates, font files, and design assets for automation. |

### Per-Episode Subfolders

| Subfolder | Contents | Naming Convention |
|-----------|----------|-------------------|
| **Scripts/** | Final script JSON, narration text, shorts breakdown | `ep{nn}-script-v{n}.json`, `ep{nn}-narration.txt` |
| **VO/** | Voice-over audio files (ElevenLabs output) | `ep{nn}-vo-full.mp3`, `ep{nn}-vo-short-{nn}.mp3` |
| **Visuals/** | AI-generated images, video clips, b-roll | `ep{nn}-scene-{nn}-{description}.png` |
| **Assembled/** | Final rendered video files (long-form) | `ep{nn}-{slug}-v{n}.mp4` |
| **Shorts/** | Rendered short-form clips (9:16) | `ep{nn}-short-{nn}-{hook}.mp4` |
| **_Archive/** | Rejected versions, superseded assets | Move here instead of deleting |

## n8n Implementation Notes

### Creating Folders Programmatically

Folders can be created via the Google Drive v3 API node in n8n:

```
Resource: File/Folder
Operation: Create
Name: {folder_name}
Parents: [{parent_folder_id}]
MimeType: application/vnd.google-apps.folder
```

### Resolving Folder IDs

After manual or programmatic creation, use this n8n pattern to resolve a folder ID by name:

```
Resource: File
Operation: List
Query: name = '{folder_name}' and mimeType = 'application/vnd.google-apps.folder' and '{parent_id}' in parents and trashed = false
```

### Storage Estimates

| Asset Type | Size per Episode | 22 Episodes Total |
|------------|-----------------|-------------------|
| Script JSON | ~50 KB | ~1.1 MB |

| Long-form Video (10-20 min, 1080p) | ~500 MB - 1.5 GB | ~11-33 GB |
| Shorts (5-8 per episode) | ~200-500 MB | ~4.4-11 GB |
| Thumbnails | ~2-5 MB | ~44-110 MB |
| **Total Estimate** | **~720 MB - 2 GB** | **~16-45 GB** |

Google Drive free tier is 15 GB. A Google One plan (100 GB for $1.99/month or 200 GB for $2.99/month) is required for this project.
