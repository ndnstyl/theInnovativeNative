# YouTube Platform Specifications

## Overview

World's second-largest search engine and largest video platform. 2.5B+ monthly active users. Best for long-form content, tutorials, brand storytelling, and video ads. Owned by Google, uses Google Ads for advertising.

### Thumbnail Best Practices

1. **Faces with emotion** - 10-20% CTR lift
2. **High contrast** - Stands out in recommendations
3. **3-5 words max** - Large, readable text
4. **Bright colors** - Yellow, red, orange perform best
5. **Consistent branding** - Recognizable style
6. **Curiosity gap** - Don't reveal everything

## Best Practices

### Video Content Strategy

1. **Hook in 0-5 seconds** - Critical for retention
2. **Pattern interrupt** - Start with something unexpected
3. **Open loop** - Create curiosity early
4. **Deliver value** - Match title/thumbnail promise
5. **Clear CTA** - Subscribe, like, comment, link

### Title Optimization

1. **Front-load keywords** - First 60 chars most important
2. **Include numbers** - "7 Ways to..." performs well
3. **Emotional triggers** - "Shocking", "Amazing", "Secret"
4. **Brackets work** - [2024], (Step-by-Step)
5. **Avoid clickbait** - Deliver on promise

### Description Optimization

1. **First 100 chars** - Hook + primary keyword
2. **Timestamps** - Improves watch time
3. **Links** - Products, socials, related videos
4. **Keywords naturally** - Don't stuff
5. **CTAs** - Subscribe, notifications, comments

### Thumbnail Rules

| Do | Don't |
|----|-------|
| Faces with emotion | Blurry images |
| High contrast | Cluttered design |
| 3-5 words max | Tiny text |
| Bright colors | Dull/muted colors |
| Consistent branding | Different style each video |
| Test variants | Set and forget |

### Posting Schedule

| Frequency | Best For |
|-----------|----------|
| Daily | Gaming, vlogs, news |
| 3x/week | Most channels |
| 1x/week | High-production content |
| Best Days | Thursday, Friday, Saturday |
| Best Times | 2pm-4pm (local time) |

## YouTube Algorithm Factors

### Main Ranking Signals

1. **Click-through rate (CTR)** - Thumbnail + title effectiveness
2. **Watch time** - Total minutes watched
3. **Average view duration** - % of video watched
4. **Engagement** - Likes, comments, shares
5. **Session time** - How long viewers stay on YouTube

### Recommended Feed

- Viewer's watch history
- Similar videos watched
- Channel subscriptions
- Topic interests

### Search Ranking

- Title relevance
- Description keywords
- Tags (minor factor)
- Watch time for query
- CTR for query

## Compliance Notes

1. **Copyright**
   - Use royalty-free music
   - Get licenses for copyrighted content
   - Content ID may claim revenue
   - Fair use is limited

2. **Ad Policies**
   - No misleading content
   - Age-appropriate content for ads
   - No controversial content adjacency

3. **Community Guidelines**
   - No spam or deceptive practices
   - No harmful or dangerous content
   - No harassment or cyberbullying
   - No hate speech
   - Strike system (3 strikes = termination)

4. **COPPA Compliance**
   - Mark content as "Made for Kids"
   - Limited features on kids content
   - No personalized ads on kids content
