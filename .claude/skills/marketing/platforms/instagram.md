# Instagram Platform Specifications

## Overview

Visual-first platform owned by Meta. 2B+ monthly active users, skewing younger (18-34). Best for brand awareness, lifestyle products, influencer marketing, and visual storytelling. Shares ad platform with Facebook.

---

## Image Specifications

### Profile Assets

| Asset | Dimensions | Ratio | Notes |
|-------|-----------|-------|-------|
| Profile Picture | 320 x 320px | 1:1 | Displays as 110px circle |
| Highlight Cover | 1080 x 1080px | 1:1 | Shows as circle |

### Feed Posts

| Format | Dimensions | Ratio | Notes |
|--------|-----------|-------|-------|
| Square | 1080 x 1080px | 1:1 | Classic, reliable |
| Portrait | 1080 x 1350px | 4:5 | Max vertical, more screen space |
| Landscape | 1080 x 566px | 1.91:1 | Less common, smaller in feed |

### Stories & Reels

| Format | Dimensions | Ratio | Notes |
|--------|-----------|-------|-------|
| Story | 1080 x 1920px | 9:16 | Full screen vertical |
| Reel | 1080 x 1920px | 9:16 | Same as Story |
| Reel Cover | 1080 x 1920px | 9:16 | Or 1:1 for grid preview |

### Carousels

| Spec | Requirement |
|------|-------------|
| Cards | 2-10 images/videos |
| Dimensions | 1080 x 1080px (square) or 1080 x 1350px (portrait) |
| Mix | Can combine images and videos |
| Ratio | All cards must be same ratio |

### Instagram Ads

| Placement | Dimensions | Ratio |
|-----------|-----------|-------|
| Feed Image | 1080 x 1080px | 1:1 |
| Feed Image (Alt) | 1080 x 1350px | 4:5 |
| Story Ad | 1080 x 1920px | 9:16 |
| Reels Ad | 1080 x 1920px | 9:16 |
| Explore Ad | 1080 x 1080px | 1:1 |
| Carousel Card | 1080 x 1080px | 1:1 |

## Character Limits

### Organic Posts

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Caption | 2,200 chars | First 125 chars visible before "more" |
| Hashtags | 30 max | 5-10 optimal (in caption or comment) |
| Bio | 150 chars | Include CTA and link |
| Username | 30 chars | Memorable, searchable |
| Alt Text | 100 chars | For accessibility |

### Instagram Ads

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Primary Text | 125 chars | Before truncation |
| Headline | 40 chars | For some placements |
| Hashtags in Ads | 30 max | 3-5 recommended |

### Stories & Reels

| Element | Limit |
|---------|-------|
| Story Text Overlay | No hard limit | Keep minimal |
| Reel Caption | 2,200 chars |
| Reel Audio Title | 100 chars |

## Video Specifications

| Spec | Requirement |
|------|-------------|

| File Size | Up to 4GB |
| Resolution | 1080p minimum |
| Aspect Ratio | 1:1, 4:5, or 16:9 |

| Format | MP4, MOV |
| Audio | AAC, 128kbps+ |

### Stories

| Spec | Requirement |
|------|-------------|
| Duration | 1-60 seconds (15s segments) |
| Resolution | 1080 x 1920px |
| Aspect Ratio | 9:16 |
| Safe Zone | Top 14% (250px) for UI, bottom 20% for CTA |

### Reels

| Spec | Requirement |
|------|-------------|
| Duration | 15, 30, 60, or 90 seconds |
| Resolution | 1080 x 1920px |
| Aspect Ratio | 9:16 |
| Audio | Use trending sounds for reach |

## Best Practices

### Content Strategy

1. **Visual Quality**
   - High-resolution images only
   - Consistent color palette/filters
   - Professional but authentic
   - User-generated content performs well

2. **Feed Aesthetics**
   - Maintain visual consistency
   - Plan grid layout
   - Alternate content types
   - Use consistent editing style

3. **Stories Strategy**
   - Post 3-7 stories per day
   - Mix formats (photos, videos, polls)
   - Use interactive stickers
   - Behind-the-scenes content

4. **Reels Strategy**
   - Hook in first 1-3 seconds
   - Use trending audio
   - Keep text on screen for muted viewers
   - End with CTA

### Engagement Tactics

1. **Caption Structure**
   - Hook in first line
   - Value in the middle
   - CTA at the end
   - Line breaks for readability

2. **Hashtag Strategy**
   - Mix of sizes (big, medium, niche)
   - Relevant to content
   - Research competitors
   - Create branded hashtag

3. **Engagement**
   - Reply to comments within 1 hour
   - Like and comment on followers' content
   - Use Stories polls/questions
   - Go Live regularly

### Posting Times

| Time Frame | Engagement |
|------------|-----------|
| 6am - 9am | Morning scroll |
| 12pm - 2pm | Lunch break |
| 7pm - 9pm | Evening peak |
| Best Day | Wednesday |

### Ad Best Practices

1. **Creative**
   - Native, non-ad look
   - User-generated style
   - Vertical video for Stories/Reels
   - Test multiple formats

2. **Copy**
   - Front-load the hook
   - Use emojis strategically
   - Clear CTA
   - Minimal hashtags in ads

3. **Targeting**
   - Use Facebook's targeting
   - Leverage Instagram engagement audiences
   - Lookalike audiences work well
   - Retarget website visitors

## Algorithm Factors (2024+)

### Feed Ranking
1. Relationship (interaction history)
2. Interest (content type preference)
3. Timeliness (recency)
4. Session (user behavior patterns)

### Reels Ranking
1. Watch time (completion rate)
2. Engagement (likes, comments, shares)
3. Audio usage (trending sounds)
4. Relevance signals

### Stories Ranking
1. View history
2. Engagement with stories
3. Closeness (DM frequency)

## Compliance Notes

1. **Branded Content**
   - Require "Paid partnership" label
   - Use Branded Content tool
   - Influencer disclosure required

2. **Ad Policies**
   - Same as Facebook
   - Extra scrutiny on beauty/health claims
   - No before/after imagery

3. **Music in Ads**
   - Licensed music library only
   - No copyrighted audio in paid content
   - Use Meta's Sound Collection
