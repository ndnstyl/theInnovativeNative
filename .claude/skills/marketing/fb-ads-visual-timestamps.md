## Key Frameworks to Visualize

1. **Market Awareness Pyramid** [1:01]
   - 3% Ready to buy
   - 17% Information gathering
   - 20% Problem aware
   - 60% Not problem aware

2. **Hyper-Dopamine Formula** [15:00, 17:30]
   - Clickbait + Targeted Benefit = Hyper-Dopamine Ad
   - Pattern Interrupt + Burning Intrigue + Big Specific Benefit

3. **Ad Anatomy** [7:01, 25:01]
   - Scroll-stopping image
   - Eyeball-grabbing headline
   - Slippery lead-in copy
   - Butter body copy
   - Intriguing link description
   - Non-threatening CTA (Learn More)

4. **Click Drivers** [5:00]
   - Curiosity (most powerful)
   - Intrigue
   - Shock
   - Direct benefit
   - Implied benefit
   - Fear
   - Vanity
   - Self-interest

5. **Creative Styles** [21:00-24:00]
   - Raw Native (iPhone photos)
   - SMS Ads
   - Breaking News
   - Native Highlight (circles/arrows)
   - Native Social Post
   - Secret Info
