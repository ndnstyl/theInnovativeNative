# SOP: Ad Copywriting

## Purpose
Standard operating procedure for writing high-converting ad copy following the Hyper-Dopamine methodology.

## Scope
- Facebook/Instagram ad copy
- Google Ads text
- LinkedIn ad copy
- YouTube ad scripts
- Headlines and hooks

## The Hyper-Dopamine Framework

### Core Formula
```
Clickbait + Targeted Benefit = Hyper-Dopamine Ad
```

### Three Elements
1. **Pattern Interrupt** - Something unexpected
2. **Burning Intrigue** - Curiosity that demands a click
3. **Big Specific Benefit** - Clear value for the target audience

---

## Copywriting Workflow

### Phase 1: Research (30 min)

1. **Understand the Audience**
   - Who are they?
   - What problem do they have?
   - What language do they use?
   - What motivates them?

2. **Study Top Performers**
   - Review gossip sites (Lad Bible, TMZ, E-News)
   - Check Facebook Ad Library for competitors
   - Save winning headlines to swipe file

3. **Identify the Angle**
   - What's the one big benefit?
   - What's the curiosity hook?
   - What's the pattern interrupt?

### Phase 2: Drafting (45 min)

1. **Write 5-10 Headlines**
   - Use swipe formulas (see below)
   - Mix different angles
   - Vary length

2. **Write Primary Text Variations**
   - Hook in first 40 characters
   - Expand the curiosity
   - Include social proof if available
   - Clear CTA

3. **Write Descriptions**
   - Reinforce the hook
   - Add urgency if appropriate

### Phase 3: Editing (30 min)

1. **Run Through Hemingway App**
   - Target grade 5 or below
   - Shorten sentences
   - Simplify words

2. **Mobile Preview**
   - Check truncation points
   - Ensure hook visible
   - Test readability

3. **Brand Voice Check**
   - Align with tone of voice
   - Check for off-brand language

## Headline Swipe Formulas

### Curiosity-Based

| Formula | Example |
|---------|---------|
| [Group] going wild after spotting [discovery] | "Aussies going wild after spotting home loan hack" |
| [Role] advised to do one thing to [benefit] | "CEOs advised to do one thing before going home" |
| Insane new [thing] leaves [audience] speechless | "Insane new mindset trick leaves anxiety sufferers speechless" |
| Secret way to [achieve] without [obstacle] | "Secret way to find funding without banks" |
| [Person/Group] discovers mystery [solution] | "Miami mom discovers mystery metabolism booster" |
| True story behind [intriguing thing] | "True story behind twisted 56kg weight loss journey" |

### Benefit-Focused

| Formula | Example |
|---------|---------|
| How to [benefit] in [timeframe] | "How to generate 90-480 leads per month" |
| [Number] ways to [benefit] | "7 ways to slash your ad costs in half" |
| The [adjective] way to [benefit] | "The lazy way to get more clients" |
| Why [common belief] is wrong | "Why cold calling is dead (and what works now)" |

### Social Proof

| Formula | Example |
|---------|---------|
| [Celebrity/Expert] reveals [secret] | "Mark Cuban reveals his hiring secret" |
| How [person] went from [before] to [after] | "How Sarah went from 0 to 50k followers" |
| [Number] [people] can't be wrong | "10,000 business owners can't be wrong" |

## Copy Rules

### DO

1. **Write at 5th grade level**
   - Use Hemingway App
   - Short words (1-2 syllables)
   - Short sentences (under 20 words)

2. **Front-load the hook**
   - Best stuff in first 40 characters
   - Assume truncation

3. **Use line breaks**
   - One thought per line
   - Easy to scan on mobile
   - Lots of white space

4. **Be specific**
   - "12-19 kilos" not "lose weight"
   - "90-480 leads" not "more leads"
   - Numbers add believability

5. **Write to one person**
   - "You" not "our customers"
   - "We" not company name
   - Personal, conversational tone

6. **Sell the click**
   - Not the product
   - Create curiosity gap
   - Promise value on the other side

### DON'T

1. **Don't look like an ad**
   - Avoid salesy language
   - Native content feel
   - No "BUY NOW!!!"

2. **Don't use blind clickbait**
   - Must include targeted benefit
   - Not just curiosity for curiosity's sake
   - Wrong: "You won't believe what Ivanka does"
   - Right: "Supermodels use these 3 tricks to look younger"

3. **Don't overuse "you/your"**
   - Platforms penalize personal callouts
   - Mix it up with indirect language

4. **Don't be vague**
   - Specificity = believability
   - Generic claims get ignored

## Platform-Specific Guidelines

### Meta (Facebook/Instagram)

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Primary Text | 125 visible | Hook in 40 chars |
| Headline | 40 chars | 27 recommended |
| Description | 30 chars | Reinforce hook |

### Google Search

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Headline | 30 chars x 15 | Include keywords |
| Description | 90 chars x 4 | Benefits + CTA |
| Display Path | 15 chars x 2 | Use keywords |

### LinkedIn

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Introductory Text | 600 chars | 150 visible |
| Headline | 200 chars | 70 recommended |
| Professional tone | Required | Still use curiosity |

### YouTube

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Video Title | 100 chars | 60 visible |
| Description | 5000 chars | Hook in first 100 |

## Copy Templates

### Facebook Ad - Lead Gen

```
[HOOK - 40 chars max]

[Expand the intrigue - 2-3 short sentences]

[Social proof or credibility]

[What they'll learn/get]

[CTA - Click below to...]
```

### Facebook Ad - Sales

```
[HOOK with benefit]

[Problem agitation - short]

[Solution tease]

[Specific results/proof]

[CTA]
```

### Google Search Ad

**Headlines:**
1. [Primary keyword + benefit]
2. [Social proof / number]
3. [Urgency / scarcity]
4. [Question]
5. [How to...]

**Descriptions:**
1. [Expand benefit + CTA]
2. [Features + differentiator]

## Review Checklist

Before submitting copy:

- [ ] Hook in first 40 characters
- [ ] Grade level 5 or below (Hemingway)
- [ ] Specific numbers/results
- [ ] Clear benefit for target audience
- [ ] Curiosity gap created
- [ ] Doesn't look like an ad
- [ ] Mobile preview checked
- [ ] Brand voice aligned
- [ ] No platform policy violations

---

## A/B Testing Guidelines

### What to Test

1. **Headlines** (highest impact)
2. **Hook/first line**
3. **Benefit angle**
4. **Social proof inclusion**
5. **CTA variation**

### Test Structure

- Test one variable at a time
- Run for minimum 1000 impressions
- Statistical significance before calling winner
- Document results in learnings file
